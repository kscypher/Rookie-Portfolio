Smart Attendance System using FRS

The Smart Attendance System Using FRS is a face recognition-based attendance system designed to automate attendance marking efficiently. Unlike traditional manual methods, which are time-consuming and error-prone, this system eliminates proxy attendance and reduces queue-based delays. It utilizes OpenCV for image capture, face_recognition for processing, and tkinter for an interactive GUI. The system stores face data using Pickle for future recognition and integrates a custom utility module for streamlined operations. This project enhances security, improves efficiency, and modernizes attendance tracking in various sectors.

Hovering Periodic Table

The Periodic Table project, developed using Vanilla JavaScript on HatchXR.com, provides an interactive experience by displaying atomic details such as name, number, weight, and family. It uses event listeners to enhance user interaction, highlighting elements when hovered over and updating relevant information in real time. The project ensures a smooth and dynamic UI by adjusting element opacity for visual effects. By leveraging JavaScript's DOM manipulation, it provides instant feedback, making the periodic table an educational and visually appealing tool.

Bounce Tales

The Bouncing Ball Game is an engaging, fast-paced browser-based arcade game developed using JavaScript and p5.js. The player controls a bar at the bottom of the canvas to bounce balls and prevent them from falling off the screen. As balls bounce off walls, the ceiling, and the bar, they duplicate at an increased speed, making the game progressively harder and increasing the score. The bar shrinks over time, adding to the challenge. Points are earned for successful bounces, and the game ends when all balls drop below the bar. Features include dynamic speed adjustments, object-oriented design with a `Ball` class, responsive controls, and smooth collision detection. Future improvements include power-ups, sound effects, mobile touch support, and visual enhancements.
